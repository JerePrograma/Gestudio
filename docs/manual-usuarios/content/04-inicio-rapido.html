La operación diaria se resume en cuatro reglas: elegir el módulo correcto, usar datos reales en producción, guardar una sola vez y conservar el historial.

1. Elegí la tarea

Alumnos para personas; Inscripciones para actividades; Cobranza para dinero recibido.

2. Revisá fechas e importes

Una fecha modifica vigencias, cargos, reportes e historial.

3. Esperá el resultado

No repitas Guardar o Registrar mientras la operación está procesándose.

4. No borres historia

Usá las acciones de baja, finalización, anulación o reversión disponibles para tu rol.
