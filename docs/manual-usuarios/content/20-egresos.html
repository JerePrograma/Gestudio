Un egreso representa una salida real de dinero. Antes de registrarlo, confirmá que el movimiento no corresponda a otro flujo, como Stock.

  1. Revisá que no exista previamente.
  2. Completá motivo, monto y método según el formulario real.
  3. Registrá una sola vez.
  4. Verificá el historial y la aparición en Caja.
No uses Egresos para ocultar diferencias. Correcciones y compras de productos deben seguir el flujo específico disponible.
