Cobranza registra dinero efectivamente recibido. El formulario permite seleccionar alumno, método, monto y aplicaciones a cargos pendientes.

  1. Confirmá el alumno.
  2. Elegí el método real.
  3. Ingresá el monto recibido.
  4. Distribuí el importe sin superar saldos.
  5. Revisá total aplicado y excedente.
  6. Registrá una sola vez.
El manual no genera pagos de práctica. Las capturas usan consultas y formularios sobre datos demo existentes para preservar la idempotencia.
